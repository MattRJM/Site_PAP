Sobre o Projeto

O Sistema de Votação Escolar é um software desenvolvido em Java com integração Firebase para modernizar eleições escolares.

Ele garante votação segura, login autenticado e contagem automática de votos. Desenvolvido como parte do Projeto de Aptidão Profissional (PAP).

Funcionalidades

Login Seguro

Acesso protegido para administradores e eleitores.

Votação Única

Controle por e-mail, garantindo apenas um voto por aluno.

Resultados Automáticos

Contagem e visualização instantânea dos votos.
